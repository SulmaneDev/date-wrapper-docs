DateWrapper.js

DateWrapper ‚Äî a tiny, human-friendly, dependency-free wrapper around the native Date object. Adds parsing, formatting, arithmetic, timezone helpers, recurrence, business-day calendars, durations, ranges, and humanized diffs ‚Äî all in one minimal library.

üöÄ Why Use DateWrapper?

JavaScript‚Äôs Date object is powerful but awkward ‚Äî inconsistent parsing, limited formatting, and timezone headaches. DateWrapper keeps native Date under the hood while providing a smooth, modern API.

  • Zero dependencies
  • Chainable, ergonomic API
  • Works in modern browsers & Node.js v20+
  • Extensible plugin system

üì¶ Installation

Download: Just grab the date-wrapper.js file and include it in your project:

<script src="date-wrapper.js"></script>

Exposes: window.DateWrapper, Duration, DateRange, BusinessCalendar

Node.js

const { DateWrapper, Duration, DateRange, BusinessCalendar } = require('./date-wrapper');

‚ö° Quick Examples

// Create + format
const d = new DateWrapper('2025-01-01T10:30:00Z');
console.log(d.format('YYYY-MM-DD HH:mm:ss')); // 2025-01-01 10:30:00

// Chainable arithmetic
const next = new DateWrapper('2025-01-01').addDays(5).addMonths(1).startOf('day');
console.log(next.format('YYYY-MM-DD')); // 2025-02-06

// Timezone formatting
console.log(d.formatWithTimeZone('en-US', { timeZone: 'America/New_York', dateStyle:'full', timeStyle:'short' }));

// Recurrence
const occ = new DateWrapper('2025-01-01').rrule('FREQ=WEEKLY;BYDAY=MO,WE;COUNT=5').all();

üß≠ API Snapshot

Constructor

  • new DateWrapper(input) ‚Äî accepts Date, string, number, or object

Format

  • format(fmt) ‚Äî e.g. YYYY-MM-DD HH:mm:ss
  • formatWithTimeZone(locale, options)

Arithmetic

  • add(), subtract(), addDays(), addMonths()
  • startOf(), endOf()

Recurrence

  • rrule(rule) ‚Üí returns occurrences matching RFC5545-lite syntax

Business Calendar

  • DateWrapper.businessCalendar(opts)
  • addBusinessDays(), isBusinessDay()

Durations & Ranges

  • new Duration({...}), Duration.fromISO('P1M3D')
  • new DateRange(start, end) ‚Üí contains(), intersect()

Utilities

  • humanizeDiff() ‚Äî returns ‚Äú3 days ago‚Äù, ‚Äúin 2 weeks‚Äù, etc.
  • isoWeek(), isoWeekYear(), fromIsoWeek()
  • moonPhase(), sunrise(), sunset()

üß© Caveats

  • RRULE is lightweight (RFC5545-lite). For full coverage, use an external library.
  • Timezone handling relies on Intl APIs.
  • Sunrise/sunset calculations are approximations.
